# Smart Resume Analyzer

An AI-powered application that analyzes resumes against job descriptions
to calculate ATS compatibility and job match score using NLP & ML.

## Features
- Resume parsing (PDF)
- Skill extraction
- Job match scoring
- ATS-style analysis

## Tech Stack
- Python
- NLP
- Scikit-learn
- Streamlit

## How to Run
pip install -r requirements.txt
streamlit run app.py
